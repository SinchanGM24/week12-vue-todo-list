# Screenshots Folder

Silakan tambahkan screenshot berikut untuk melengkapi dokumentasi:

1. **tampilan-kosong.png** - Tampilan awal aplikasi tanpa tugas
2. **form-input.png** - Screenshot saat mengisi form input
3. **daftar-tugas.png** - Screenshot dengan beberapa tugas
4. **hover-effect.png** - Screenshot efek hover pada tombol/item

## Cara Mengambil Screenshot

1. Jalankan aplikasi dengan `npm run dev`
2. Buka browser dan akses `http://localhost:5173`
3. Gunakan tool screenshot (Windows: Win + Shift + S, macOS: Cmd + Shift + 4)
4. Simpan screenshot di folder ini dengan nama yang sesuai
